
# Robyn Landing Page – GitHub Pages

This folder contains:
- `index.html` – your interactive landing page
- `README.md` – quick instructions

## How to Publish on GitHub Pages

1. Go to https://github.com and sign in (or create a free account).
2. Click the **+** in the top-right → **New repository**.
3. Name it something like: `socal-apartment-seller-guide` and keep it **Public**.
4. After it’s created, click **Add file → Upload files**.
5. Upload both `index.html` and `README.md` from this zip, then **Commit changes**.
6. Go to **Settings → Pages** (left side menu).
7. Under **Source**, choose **Deploy from a branch**.
8. For **Branch**, choose `main` (or `master`) and folder `/ (root)`, then click **Save**.

GitHub will build your site. After a short time, your page will be live at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

Example:
`https://robynwonnell.github.io/socal-apartment-seller-guide/`
